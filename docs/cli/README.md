# Yard CLI Command Gallery

This section documents Yard through task-oriented scenarios. Examples use `yard` as the canonical product command.

When running from this source tree, replace `yard` with:

```bash
go run ./cmd/yard
```

## Scenarios

- [Full CLI smoke test](./smoke-test.md): validate config loading, the project registry, Lima, a remote SSH target simulated with OrbStack, and cleanup.

## Status Labels

- `Available`: implemented and covered by automated checks.
- `Experimental`: implemented, but still useful to validate in real environments.
- `Planned`: not implemented yet.
